## Definition
A **basis** for a vector space is a set of vectors that:
1. **Span** the space (you can build any vector from them), and
2. Are **linearly independent**
### Example in $\mathbb{R}^2$
Standard basis:
$$
e_0 = \begin{bmatrix} 1 \\ 0 \end{bmatrix},
\quad
e_1 = \begin{bmatrix} 0 \\ 1 \end{bmatrix}
$$
So that any vector can be written as, for example:
$$
v = 
\begin{bmatrix} 3 \\ 2 \end{bmatrix} = 
3e_1 + 2e_2
$$
So the **coordinates** of $v$ in this basis is $(3, 2)$
